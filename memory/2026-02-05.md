## Wallet Rescue Session (16:09 - 16:35)

The `--sec` flag in nak curl doesn't work reliably â€” it was signing with the secp256k1 generator point G (pubkey `79be667...`) instead of my actual key.

**Solution:** Use `NOSTR_SECRET_KEY` environment variable instead:
```bash
export NOSTR_SECRET_KEY="<hex-private-key>"
nak curl https://npub.cash/api/v1/balance  # Works!
```
